# BIST Radar Pro v14.2 - Yapay Zekâ Karar Motoru

Bu sürüm v14.0 temel arayüzün üzerine AI karar motoru ekler.

## Eklenenler
- Trend / Para Girişi / Momentum / Formasyon / Risk puanlama
- ADX, SuperTrend, RSI, MACD ve hacim kalitesi yorumları
- Karar ağacı: neden AL / İZLE / BEKLE / RİSKLİ
- Senaryo analizi: olumlu, nötr, olumsuz olasılıklar
- Pozisyon planı: hedef, stop, risk-getiri ve vade
- En Güçlü 20, Güvenli Kazançlar ve Fırsat Avcısı listeleri
- API yoksa demo veriyle çalışır, API varsa `/api/stock` yanıtını kullanır
