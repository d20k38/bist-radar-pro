# BIST Radar Pro v2.0 Production

Bu sürüm gerçek proje tabanı üzerinde hazırlanmıştır.

## Ana değişiklikler
- Arayüz sadeleştirildi: Radar, Analiz, Cockpit, Portföy, Risk, AI Açıklama, Karşılaştırma, Sistem.
- Modüllerin alt alta birikmesini engelleyen tek aktif workspace yapısı eklendi.
- Institutional Risk Engine eklendi.
- Risk Engine; likidite, para akışı, trend güveni, volatilite, manipülasyon riski, haber riski ve veri sağlığı katmanlarını tek panelde gösterir.
- R28A Cockpit, R20 Explainable AI, R21 Portfolio Intelligence ve v1.1 Snapshot Intelligence korunmuştur.
- Tek API yapısı korunmuştur: api/core.js
- Demo/random veri eklenmemiştir.
