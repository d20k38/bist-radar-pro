# BIST Radar Pro v12.1 Mobil Hibrit

Bu sürüm v12 üzerine düzenlendi:

- Mobil uyumlu arayüz: telefon ekranında kartlar, sekmeler, tablolar ve grafikler daha okunaklıdır.
- Mynet anlık fiyat doğrulaması: canlı fiyat Yahoo teknik kapanıştan aşırı farklıysa kullanılmaz, ekranda uyarı gösterilir.
- Yahoo geçmiş grafik verisi teknik analiz için korunur.
- Portföy yönetimi LocalStorage ile devam eder.

Kurulum: ZIP içindeki tüm dosya ve klasörleri GitHub repo kök dizinine yükleyin: `api/`, `lib/`, `data/`, `index.html`, `package.json`, `vercel.json`.
